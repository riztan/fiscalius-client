Manual del Usuario

Guía completa para usar Fiscalius de manera efectiva.

Introducción

Bienvenido al manual de usuario de Fiscalius. Este manual te guiará a través de todas las funcionalidades del sistema de facturación electrónica.

1. Primeros Pasos

1.1 Iniciar Sesión

Para iniciar sesión en Fiscalius:

  1. Abre la aplicación Fiscalius
  2. Ingresa tu usuario
  3. Ingresa tu contraseña
  4. Selecciona la empresa (si tienes acceso a múltiples empresas)
  5. Haz clic en Iniciar Sesión

1.2 Interfaz Principal

Una vez iniciar sesión, verás:

  • Menú principal: Acceso a todos los módulos
  • Barra de estado: Información del usuario y empresa
  • Área de trabajo: Donde se muestran los formularios

2. Módulo de Ventas

2.1 Facturación

Para emitir una factura:

  1. Ve a Ventas → Facturas
  2. Haz clic en Nueva Factura
  3. Selecciona el cliente
  4. Agrega productos con cantidades y precios
  5. Verifica impuestos (IVA, IGT)
  6. Haz clic en Guardar

2.2 Notas de Crédito

Para emitir una nota de crédito (devolución o descuento):

  1. Ve a Ventas → Notas de Crédito
  2. Haz clic en Nueva Nota
  3. Selecciona la factura original
  4. Especifica el motivo
  5. Confirma el monto
  6. Haz clic en Guardar

2.3 Notas de Débito

Para emitir una nota de débito (cargos adicionales):

  1. Ve a Ventas → Notas de Débito
  2. Haz clic en Nueva Nota
  3. Selecciona el cliente
  4. Especifica el concepto y monto
  5. Haz clic en Guardar

3. Módulo de Inventario

3.1 Catálogo de Productos

Para gestionar productos:

  1. Ve a Inventario → Productos
  2. Agregar producto: Haz clic en Nuevo
  3. Editar producto: Selecciona y haz clic en Editar
  4. Eliminar producto: Selecciona y haz clic en Eliminar

3.2 Control de Existencias

El sistema controla automáticamente las existencias cuando:

  • Se emite una factura
  • Se emite una nota de crédito
  • Se realiza una entrada de inventario
  • Se realiza una salida de inventario

4. Módulo de Finanzas

4.1 Cajas Registradoras

Para gestionar cajas:

  1. Abrir caja: Selecciona la caja y haz clic en Abrir
  2. Cerrar caja: Selecciona la caja abierta y haz clic en Cerrar
  3. Arqueo: Realiza el conteo de efectivo al cerrar

4.2 Cobranzas

Para registrar cobranzas:

  1. Ve a Finanzas → Cobranzas
  2. Haz clic en Nuevo Recibo
  3. Selecciona el cliente
  4. Selecciona las facturas a cancelar
  5. Selecciona el medio de pago
  6. Haz clic en Guardar

5. Reportes

5.1 Reportes Disponibles

  • Libro de Ventas: Resumen de ventas por período
  • Libro de Compras: Resumen de compras por período
  • Cuentas por Cobrar: Clientes con saldos pendientes
  • Arqueo de Caja: Movimientos de caja
  • Inventario: Stock actual de productos

5.2 Generar Reportes

  1. Ve a Reportes
  2. Selecciona el tipo de reporte
  3. Configura los filtros (fechas, clientes, etc.)
  4. Haz clic en Generar
  5. Imprime o exporta el reporte

6. Configuración

6.1 Datos de la Empresa

Para configurar los datos de tu empresa:

  1. Ve a Configuración → Empresa
  2. Actualiza los datos fiscales (RIF, dirección, teléfono)
  3. Configura los datos del SENIAT
  4. Haz clic en Guardar

6.2 Usuarios

Para gestionar usuarios:

  1. Ve a Configuración → Usuarios
  2. Agregar usuario: Haz clic en Nuevo
  3. Editar usuario: Selecciona y haz clic en Editar
  4. Asignar permisos: Configura los módulos accessibles

7. Atajos de Teclado

  • F5: Actualizar
  • F9: Guardar
  • F12: Cancelar
  • Ctrl+B: Buscar
  • Ctrl+N: Nuevo registro
  • Ctrl+E: Editar registro
  • Ctrl+D: Eliminar registro

8. Solución de Problemas

8.1 No puedo iniciar sesión

  • Verifica que tu usuario y contraseña sean correctos
  • Verifica que el servidor esté disponible
  • Contacta al administrador del sistema

8.2 No puedo emitir facturas

  • Verifica que tengas permisos de facturación
  • Verifica que la serie fiscal esté configurada
  • Verifica que el cliente esté registrado

8.3 El sistema va lento

  • Verifica tu conexión a Internet
  • Reinicia la aplicación
  • Contacta al soporte técnico
