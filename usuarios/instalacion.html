Guía de Instalación

Sigue estos pasos para instalar Fiscalius en tu computadora y comenzar a facturar.

Requisitos del Sistema

Antes de instalar, asegúrate de tener:

  • Sistema Operativo: Windows 7/8/10/11 o Linux (Ubuntu 18.04+, Debian 9+)
  • Conexión a Internet: Necesaria para la activación inicial y actualizaciones
  • Servidor: Acceso al servidor Fiscalius (proporcionado por tu distribuidor o administrador)
  • Espacio en disco: Mínimo 500 MB disponibles
  • Memoria RAM: Mínimo 2 GB recomendados

Pasos de Instalación

Paso 1: Descargar el Instalador

Descarga la última versión de Fiscalius desde:

  • Releases en GitHub
  • O solicítalo a tu distribuidor autorizado

Paso 2: Ejecutar el Instalador

Windows:

Haz doble clic en el archivo .exe descargado
Sigue las instrucciones del asistente
Selecciona la carpeta de instalación (recomendado: C:\Fiscalius)

Linux:

chmod +x fiscalius-instalador
sudo ./fiscalius-instalador

Paso 3: Configuración Inicial

Al abrir Fiscalius por primera vez:

  1. Ingresa los datos del servidor (IP o dominio y puerto)
  2. Ingresa tus credenciales de acceso (usuario y contraseña)
  3. Selecciona la empresa con la que trabajarás
  4. Configura tus datos fiscales (RIF, dirección, teléfono)

Paso 4: Verificar Conexión

El sistema verificará automáticamente la conexión con el servidor. Si todo está correcto, verás el mensaje "Conexión exitosa".

Primeros Pasos

Una vez instalado, te recomendamos:

  1. Configurar tu catálogo de productos
  2. Registrar tus clientes
  3. Configurar tus cajas registradoras
  4. Emitir tu primera factura

Soporte

Si tienes problemas durante la instalación:

  • Consulta nuestra sección de preguntas frecuentes
  • Abre un ticket en GitHub Issues
  • Contacta a tu distribuidor autorizado
