Preguntas Frecuentes

Resuelve tus dudas sobre Fiscalius y la facturación electrónica en Venezuela.

General

¿Es gratuito Fiscalius?

El código del cliente es 100% abierto bajo licencia MIT. Cualquier persona puede descargarlo, estudiarlo y mejorarlo sin costo. El acceso al servidor, que es donde se procesan las facturas y se almacenan los datos, es un servicio de pago que puedes contratar a través de un distribuidor autorizado.

¿Qué es Fiscalius?

Fiscalius es un sistema de facturación electrónica de código abierto, diseñado para cumplir con los requisitos del SENIAT en Venezuela. Permite emitir facturas, notas de crédito, notas de débito y ordenes de entrega de manera legal y segura.

¿Quién está detrás de Fiscalius?

Fiscalius es desarrollado por un equipo con más de 20 años de experiencia en el desarrollo de sistemas ERP. El proyecto está en sus primeras etapas y está construyendo una comunidad de usuarios, distribuidores y desarrolladores.

¿Necesito conexión a Internet?

Sí, necesitas conexión a Internet para:

  • Conectarte al servidor Fiscalius
  • Validar documentos con el SENIAT
  • Recibir actualizaciones del sistema

Acceso al Servidor

¿Cómo obtengo acceso al servidor?

El acceso al servidor se gestiona a través de distribuidores autorizados. Ellos evaluarán tus necesidades y te ofrecerán la mejor opción, ya sea un servidor en la nube (remoto) o uno instalado en tus propias instalaciones.

¿Qué incluye el servicio de servidor?

El servidor es donde se procesan las facturas y se almacenan los datos. Incluye la infraestructura necesaria para que el sistema funcione de manera segura y confiable.

Instalación y Configuración

¿En qué sistemas operativos funciona?

Fiscalius funciona en:

  • Windows 7, 8, 10 y 11
  • Linux (Ubuntu 18.04+, Debian 9+, y otras distribuciones)

¿Cómo instalo Fiscalius?

Estamos preparando un instalador que estará disponible en GitHub una vez se complete la homologación. Mientras tanto, puedes contactar a un distribuidor autorizado para obtener asistencia con la instalación.

¿Cuántas computadoras puedo instalar?

Puedes instalar Fiscalius en tantas computadoras como necesites. El sistema es multi-usuario y varios usuarios pueden trabajar simultáneamente.

¿Cómo configuro el servidor?

La configuración del servidor la realiza tu administrador o distribuidor. Necesitarás:

  • IP o dominio del servidor
  • Puerto de conexión (por defecto: 2942)
  • Credenciales de acceso (usuario y contraseña)

Uso Diario

¿Cómo emito una factura?

Para emitir una factura:

  1. Ve al menú Ventas → Facturas
  2. Haz clic en Nueva Factura
  3. Selecciona el cliente
  4. Agrega los productos con cantidades y precios
  5. Verifica los totales e impuestos
  6. Haz clic en Guardar

¿Cómo anulo una factura?

Para anular una factura, emite una Nota de Crédito asociada a la factura original. Esto mantendrá un registro completo de la transacción.

¿Cómo cierro la caja?

Al final del día:

  1. Ve a Finanzas → Cajas
  2. Selecciona la caja abierta
  3. Haz clic en Cerrar Caja
  4. Realiza el arqueo de efectivo
  5. Confirma el cierre

Soporte

¿Dónde obtengo soporte?

El proyecto está estableciendo los canales de soporte necesarios para asistir a usuarios, distribuidores y desarrolladores. Por ahora, puedes:

  • GitHub Issues - Para reportar errores o solicitar funciones
  • Contactar a un distribuidor autorizado - Para soporte personalizado
  • Consultar esta documentación - Para guías y tutoriales

¿Cómo reporto un bug?

Para reportar un bug:

  1. Ve a GitHub Issues
  2. Haz clic en New Issue
  3. Describe el problema con detalle
  4. Incluye capturas de pantalla si es posible
  5. Especifica tu versión de Fiscalius y sistema operativo

¿Cómo solicito una nueva función?

Para solicitar una nueva función:

  1. Ve a GitHub Issues
  2. Haz clic en New Issue
  3. Selecciona Feature Request
  4. Describe la función que necesitas
  5. Explica por qué sería útil
